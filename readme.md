# Tic-Tac-Toe Terminal Game

---

## Description
Hello and welcome to my project, a Tic-Tac-Toe terminal game made using only python. I started this project to finish the lessons on my Computer Science course on CodeCademy and also to familiarize myself with what I have learned over the months. The code's readability is not good right now at the moment but I am satisfied with what I have made so far.

---

## How to play
1. Find an application to run the file
    - if you are going to use the CMD, change the directory where the file is located and type 'tic-tac-toe.py'
    - if you are going to use an IDE like Visual Studio Code, just run the file and use the built in terminal of that IDE
2. Choose an opponent
    - as of this moment, 'Computer' opponent is only available
3. Choose a difficulty
    - as of this moment, 'Easy' mode is only available
4. Play and enjoy the game
